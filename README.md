# mini-reseau
application web de type réseau social, l'objectif est de maîtriser l'architecture complète d'un site web (Full Stack) : de la gestion de base de données relationnelle (MySQL) au serveur backend (Flask) jusqu'à l'interface utilisateur dynamique (JS)."

##  Technologies utilisées

* **Langage Backend :** Python 3
* **Framework Web :** Flask
* **Base de données :** MySQL
* **Frontend :** HTML5, CSS3, JavaScript
